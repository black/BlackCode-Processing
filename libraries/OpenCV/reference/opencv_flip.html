Name flip()
Examples 
import hypermedia.video.*;

OpenCV opencv;


void setup() {

    size( 640, 480 );

    opencv = new OpenCV(this);
    opencv.capture( 320, 240 );

}

void draw() {

    opencv.read();
    opencv.flip( OpenCV.FLIP_BOTH ); // flip vertically and horizontally
    opencv.absDiff();

    // display images
    image( opencv.image(OpenCV.SOURCE), 0, 0 );
    image( opencv.image(), 320, 0 );
}
Description Flip the current image around vertical, horizontal or both axes.
Syntax flip(mode);
Parameters
mode int : the used axis, FLIP_HORIZONTAL, FLIP_VERTICAL or FLIP_BOTH
Return None
